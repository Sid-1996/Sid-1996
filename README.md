## Philosophy

我想做的不是把功能堆滿，而是把多餘操作拿掉。

每個工具都來自同一個出發點：

- 減少重複
- 降低操作負擔
- 保留真正重要的體驗

> 工具只是手段，真正的價值在於它是否讓事情變得更簡單。
