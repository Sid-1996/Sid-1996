# 👋 嗨，我是 Sid

一個喜歡製作遊戲工具的作者。  

我不做作弊工具，也不喜歡破壞遊戲平衡。
相反地，我專注在：

🔹 化繁為簡 → 減少不必要的手部操作

🔹 半自動化 → 熱鍵一按，繁瑣步驟瞬間搞定

🔹 更舒適的體驗 → 輕鬆玩遊戲，不再被操作卡住

對我來說，遊戲的樂趣應該在於享受，而不是被操作的繁瑣困住。我的工具只是想幫助大家玩得更輕鬆、更舒適。 ✨
